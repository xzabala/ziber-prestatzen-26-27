GELAKO SAREKO IP BANAKETA
==========================

IP banaketa:

- DHCP helbide multzo bat izango da Wifi bidezk kontektatzen direnentzako bakarrik (beste gailu guztiei Ip finkoa izan behar du)
- Ikasle talde bakoitzak IP helbide multzoa izango du. Taldearen baitan adostuko da zeintzuk izango diren zerbitzu edo sare gailuentzako eta zeintzuk erabiltzaileentzako.
- Irakasle bakoitzak IP helbide multzoa izango du
- Gateway helbidea: X.X.X.254
